This is a full raw flash (bootloader + partitions + app) — it fully erases and reinstalls firmware. It is not a signed OTA update; only use it for bringing an old device up to the point where it can accept signed updates, per shared/docs/OTA_RELEASE_PROCESS.md. After this flash, use the normal signed-OTA path (commissioning station or the device's own "Check for Updates" button) for anything after this point.
After it finishes: if your USB–UART adapter doesn't have RTS wired, the page can't auto-reset the board, so it may stay in download mode. Press the physical EN / RESET button (BOOT button not held) to boot the new firmware, then check the serial monitor for the new version number.
